# tw-border

A collection of Tailwind CSS v4 utilities for advanced border customization and animations

## Installation

Install the plugin:
```bash
npm install -D tw-border
```

Add the `@import` directive for this plugin to your `app.css` or `globals.css` file:
```css
@import "tw-border";
```
